# Welcome to My GitHub Pages Site
https://bdcom-dragon.github.io

This is a simple static site hosted using GitHub Pages. You can use this site to share your projects, blogs, or any other content you want to showcase.

## Features
- Easily customizable layout
- Supports markdown for documentation
- Free hosting on GitHub

## Getting Started
To start using this site:
1. Clone the repository.
2. Modify the files in the `docs` folder to update content.
3. Commit your changes and push to GitHub.

For more detailed documentation on how to set up GitHub Pages, visit the official [GitHub Pages documentation](https://pages.github.com/).  

Happy coding!
